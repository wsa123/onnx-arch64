## 生成 GitHub Actions 工作流：交叉编译 ONNX Runtime 1.15 for AArch64

- **任务**：生成使用 Linaro GCC 7.5.0 (`gcc-linaro-7.5.0-2019.12-x86_64_aarch64-linux-gnu`) 交叉编译 ONNX Runtime 1.15.1 的 GitHub Actions 工作流。
- **产物文件**：`.github/workflows/build-onnxruntime-aarch64.yml`
- **关键配置要点**：
  - Runner：`ubuntu-22.04`
  - 工具链下载来源：Linaro 官方 releases 页面（`releases.linaro.org`）
  - CMake 工具链文件：在 CI 步骤中动态生成，指定 sysroot 为 Linaro 包内的 `aarch64-linux-gnu/libc`
  - CMake 关键选项：`-Donnxruntime_CROSS_COMPILING=ON`、关闭所有 EP（CUDA/TensorRT/NNAPI/XNNPACK）、关闭 Python binding 与 unit test
  - 构建完成后验证 `.so` 文件架构为 AArch64，打包为 `.tar.gz` 并上传 Artifact
  - Tag 触发时自动发布 GitHub Release
